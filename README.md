# ML Course Homeworks 2026

This repository contains my homeworks for the Machine Learning course.

## Structure

Each homework is stored in a separate folder:

- hw01_setup_tools - Настройка инструментов и первый ноутбук
- hw02_eda - Исследовательский анализ данных (EDA) для датасета Spotify 2023
- hw03_knn - Классификация с использованием kNN на датасете Breast Cancer Wisconsin
- hw04_linear_regression_lecture - Линейная регрессия на датасете Toyota Corolla
- hw05_linear_regression_practice - Практика по линейной регрессии
- hw06_logistic_regression_theory - Логистическая регрессия (теория)
- hw07_logistic_regression_practice - Логистическая регрессия (практика)
- hw08_naive_bayes - Наивный Байес
- hw09_decision_trees - Деревья решений
